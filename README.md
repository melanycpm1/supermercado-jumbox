## Jumbox – Plataforma de Gestión de Inventarios en Supermercados

Jumbox es una plataforma integral para **gestionar inventarios, ventas y pedidos** en una cadena de supermercados.  
Permite controlar el stock de productos, registrar ventas, ingresar nuevos artículos, realizar pedidos entre sucursales, gestionar compras a proveedores y generar estadísticas centralizadas para toda la empresa.

---

##  Descripción del Proyecto
La base de datos y la aplicación están diseñadas para administrar las operaciones de una empresa como **Jumbox**, que puede poseer **varias sucursales** e incluso otras cadenas de supermercados adquiridas.  
El sistema permite:
- Mantener actualizado el inventario de cada sucursal.
- Coordinar pedidos entre sucursales para evitar faltantes de stock.
- Registrar compras a proveedores externos 
- Generar reportes y estadísticas para la toma de decisiones.
